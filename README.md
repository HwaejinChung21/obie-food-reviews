# Obie Food Reviews

A mobile app for rating and reviewing dining hall food at Oberlin College. Users can browse daily menus, leave ratings and reviews on menu items, and see what others have posted. Sign-up is restricted to @oberlin.edu emails.

Demo: https://obie-food-reviews.expo.app

## Tech Stack

**Mobile:** React Native (Expo), TypeScript, Expo Router, NativeWind, Supabase Auth

**Backend:** NestJS, PostgreSQL (Supabase), JWT authentication

**Data**
- Automated menu ingestion from AVI Dining Services
